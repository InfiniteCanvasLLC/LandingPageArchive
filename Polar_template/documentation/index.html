A) Installation - top

1. Download Polar from Themeforest

Download Polar Template from Themeforest and unzip it with any zip program or 7zip. Once unzipped, you will find these file in unzipped folder.

  • polar-template.zip - zip of template files that need to be uploaded to your root directory.
  • license - license of the theme.
  • documentation - documentation files in html format.

2. Setup the template on Server

Before install the theme, make sure your host has PHP 5 or higher (required by Swift Mailer Libs).

Install via FTP

You need an FTP program such as FileZilla to upload the template package to server.

  1. Unzipped the polar-template.zip.
  2. Upload all file in upzipped folder to your root directory with FileZilla or other FTP client.
